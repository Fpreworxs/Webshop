# Preworxs Webshop - Inventum Modul-AIR Green Comfort

A B2B webshop for selling Inventum heat pump appliances, built with HTML, CSS, and JavaScript.

## Features

- Product image carousel with 8 rotation views
- Configuration options: Model, Liters (100/150), Pipe ends (Left/Right)
- Shopping cart with quantity tracking per configuration
- Order form with email submission
- Responsive design based on Preworxs.nl styling

## Live Demo

Visit: https://YOUR_USERNAME.github.io/preworxs-webshop/

## Setup

1. Clone this repository
2. Open `index.html` in a browser

## Email Configuration

To enable email notifications, configure EmailJS in `script.js`:

1. Create a free account at https://www.emailjs.com/
2. Add an email service and create a template
3. Update the configuration at the top of `script.js`:
   - `EMAILJS_PUBLIC_KEY`
   - `EMAILJS_SERVICE_ID`
   - `EMAILJS_TEMPLATE_ID`
